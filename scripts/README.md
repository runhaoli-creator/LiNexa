# scripts

Shell entry points. All scripts are placeholders for now.

- `setup_submodules.sh` — init/update submodules under `extern/`.
- `apply_patches.sh` — apply patches from `patches/` onto the corresponding submodule.
- `run_baseline.sh` — baseline run (no TTT).
- `run_open_loop_ttt.sh` — open-loop TTT eval.
- `run_simple_ttt.sh` — closed-loop TTT on SIMPLE.
