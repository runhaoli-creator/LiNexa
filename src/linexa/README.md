# src/linexa

Project source code. Submodules under `extern/` are imported via their own packaging; this package only contains LiNexa-specific code.

- `ttt/` — episode-level fast-weight adaptation logic.
- `adapters/` — wrappers around Psi0 / SIMPLE interfaces.
- `eval/` — open-loop and closed-loop evaluation.
- `utils/` — shared utilities (seeding, logging, io).
- `cli/` — command-line entry points.
